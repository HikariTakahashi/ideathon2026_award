# 三団体合同アイデアソン 賞状HTML

このリポジトリは、三団体合同アイデアソンで受賞チームへ送付する賞状HTMLを管理するためのものです。

各賞状は、メールで送付することを想定し、外部CSSや画像に依存しないHTMLとして作成しています。受賞チーム名や日付、認定団体名などを必要に応じて編集し、メール本文への貼り付けやHTMLファイルとしての共有に利用できます。

## アイデアソンについて

本アイデアソンは、参加者が身近な課題や社会の中にある不便に目を向け、それを解決するためのプロダクトアイデアを考え、形にすることを目的としたイベントです。
限られた時間の中で、チームごとに課題の発見、ユーザー視点での検討、機能提案、実現可能性の整理、将来展望の発表までを行います。完成度だけではなく、課題に向き合う姿勢や、今後の成長につながる挑戦も大切な評価対象としています。

## 審査項目

本大会では、以下の観点をもとに審査を行います。

- ユーザー目線を持ったプロダクトになっているか
- 課題解決に適切な機能が提案できているか
- プロダクトとして実現可能であるか
- 将来の展望が課題解決に適切か

## 賞について

### 最優秀賞

総合的に最も高く評価されたチームへ贈る賞です。ユーザー目線、課題解決性、実現可能性、将来性の各観点において優れた成果を示し、本大会を代表する取り組みとして認められたチームを表彰します。

対象ファイル: `GrandPrize.html`

### 優秀賞

審査項目において高い評価を得たチームへ贈る賞です。課題の捉え方やプロダクトの提案内容に優れた点があり、今後の発展も期待できる取り組みを表彰します。

対象ファイル: `ExcellenceAward.html`

### スポンサー賞

スポンサー企業・団体の視点から、特に魅力や可能性が感じられたチームへ贈る賞です。大会全体の審査観点に加え、スポンサーごとの価値観や注目ポイントに沿って選出されます。

対象ファイル:

- `SponsorAward_TechTrain.html`
- `SponsorAward_Wealthnavi.html`

### ウィング賞

最優秀賞、優秀賞、スポンサー賞には惜しくも届かなかったものの、運営側が強く可能性を感じたチームへ贈る賞です。

本賞には、挑戦を称える意味に加え、初心者の参加者やこれから経験を積んでいくチームに、今後さらに成長し、新たな舞台へ羽ばたいてほしいという願いを込めています。

対象ファイル: `WingAward.html`

## ファイル一覧

- `GrandPrize.html`: 最優秀賞の賞状
- `ExcellenceAward.html`: 優秀賞の賞状
- `SponsorAward_TechTrain.html`: TechTrainスポンサー賞の賞状
- `SponsorAward_Wealthnavi.html`: WealthNaviスポンサー賞の賞状
- `WingAward.html`: ウィング賞の賞状

## 利用方法

1. 利用したい賞のHTMLファイルを開きます。
2. 受賞チーム名、認定者名、団体名など、必要な箇所を編集します。
3. HTMLメール本文として利用するか、HTMLファイルとして受賞者へ共有します。
